SSD 外接硬碟開箱與速度測試

M.2 SSD 搭配外接盒組一個 SSD 外接硬碟，說明安裝、格式化與測速結果

2024/02/23
閱讀時間: 3 分鐘

引言

買的 M.2 NVMe 外接硬碟盒終於到貨了。

本篇將進行說明安裝與測速，以供後人參考。

最終目的為組一個 USB 3.2 Gen 2 的 SSD 外接硬碟。

外接硬碟盒

延續前篇 M.2 SSD 外接固態硬碟選購指南

最終根據外觀與需求選擇了 iTGZ 外接盒，使用的是瑞昱的 RTL9210。

本次在淘寶上含運費約 ¥65，約為台幣 295 元。

整個內容物包含：

  1. M.2 外接硬碟盒。
  2. 十字螺絲起子。
  3. 20 公分傳輸線，Type C 對 C 與 Type C 對 A 各一。
  4. 散熱片。

M.2 SSD

本身 RTL9210 支援 PCIe Gen3×2，其設計為 USB 3.2 Gen 2，根據這兩點只需要選擇 PCIe 3 即可。

在此選擇了 TOSHIBA 旗下的KIOXIA EXCERIA G2 1TB，其特點為：

  1. 讀寫都大於 1200 MB/s。
  2. 主控：Phison E12C。
  3. NAND 類型：TLC。
  4. 便宜。

2024/02/01 購入，南紡 3C 販售價格為 NT$1590，搭配折價券在折 200 元。實付 NT$1390。

安裝 M.2 SSD 到外接盒

安裝 SSD 到外接盒非常容易：

  1. 透過外接盒所附贈的附螺絲起子，把外接盒打開。
  2. 插入 M.2 SSD。
  3. 貼上散熱片。
  4. 蓋上外接盒蓋子並鎖上螺絲。

並使用附贈的線材連接至電腦。

格式化與 4K 對齊外接 SSD

預設插入電腦，電腦是不會出現裝置的，因為沒有格式化。

首先點選鍵盤 Win + R 叫出執行，輸入 diskpart，

輸入 list disk 顯示當前電腦的硬碟資訊：

DISKPART> list disk

  磁碟 ###  狀態           大小     可用     Dyn  Gpt
  --------  -------------  -------  -------  ---  ---
  磁碟 0    連線              953 GB  1024 KB        *
  磁碟 1    連線               14 GB      0 B
  磁碟 2    連線              931 GB   931 GB

在這台電腦，硬碟 2為插入的 SSD 外接盒。透過 select disk 2 選擇這個硬碟：

DISKPART> select disk 2

磁碟 2 是所選擇的磁碟。

建立主要分割區，其中 4K 進行對齊：

DISKPART> create partition primary align=4096

DiskPart 成功地建立了指定的磁碟分割。

輸入format fs=exFAT quick進行格式化，其檔案格式為 exFAT，可在 Windows、macOS、Android、iOS 上進行存取。

DISKPART> format fs=exFAT quick

  100 % 已完成

DiskPart 已順利格式化磁碟區。

輸入list vol顯示磁碟區的標籤，可以看到磁碟區 3 還沒有標籤。

DISKPART> list vol

  磁碟區 ###  Ltr  標籤         Fs     類型        大小     狀態       資訊
  ----------  ---  -----------  -----  ----------  -------  ---------  --------
  磁碟區 0     C                NTFS   磁碟分割         953 GB  良好         開機
  磁碟區 1                      FAT32  磁碟分割         100 MB  良好         系統
  磁碟區 2                      NTFS   磁碟分割         683 MB  良好         隱藏
  磁碟區 3                      exFAT  磁碟分割         931 GB  良好

選擇該磁碟區sel vol 3，並指派一個磁碟編號，在此設定為 E 槽，其指令為assign letter=E：

DISKPART> select vol 3

磁碟區 3 是所選擇的磁碟區。

DISKPART> assign letter=E

DiskPart 成功地指派了磁碟機代號或掛接點。

打開本機，就會看到外接硬碟了。

其中可以看到的是，順序讀寫約為 900 MB/s 附近。

結論

以 NT$1685 的價格組好一個 USB 3.2 Gen 2 的 1 TB SSD 外接硬碟覺得 CP 值蠻高的。

後續就看未來使用上有什麼問題再繼續補充了。
